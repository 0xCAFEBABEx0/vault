# Designing the hf CLI as an agent-optimized way to work with the Hub

## Source Metadata

- source:: [Hugging Face Blog](https://huggingface.co/blog/hf-cli-for-agents)
- published:: 2026-06-04

## Abstract / Summary
Hugging Face describes how the official hf CLI was redesigned for coding agents as well as humans. The CLI now detects agent environments, avoids interactive blockers, emits more parseable output, preserves complete values, and provides actionable hints and skill references. Benchmarks in the post argue that using the hf CLI can reduce token usage and tool calls for complex Hub tasks compared with agents hand-rolling curl calls or using the Python SDK directly.
