## Overview
EVA-Bench Data 2.0 is an expanded benchmark for evaluating multimodal agents across diverse tool-use scenarios.

## Key Additions
- 3 major domains expanded to 121 tools and 213 scenarios
- More realistic, complex multi-step tasks
- Better coverage of real-world tool use patterns

## Domains
- Information retrieval and synthesis
- Code generation and execution
- Creative content production

## Evaluation
- Success rate, efficiency, and error analysis
- Comparison across frontier multimodal models
- Detailed per-domain performance breakdowns

## Significance
Sets a new standard for evaluating agent capabilities in practical tool-use scenarios, driving progress toward more capable AI assistants.
