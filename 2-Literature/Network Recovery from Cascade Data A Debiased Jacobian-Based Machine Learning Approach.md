## Overview
A machine learning approach for recovering network structure from cascade data using debiased Jacobian estimation. The method addresses the challenge of inferring graph topology from observed diffusion processes.

## Key Contributions
- Debiased Jacobian-based estimator for network recovery
- Theoretical analysis of estimation bias in cascade data
- Scalable ML framework for large-scale network inference

## Method
- Modeling cascade dynamics on unknown networks
- Jacobian computation to capture sensitivity of cascades to network structure
- Debiasing procedure to correct for systematic estimation errors
- Integration with modern graph neural network architectures

## Results
- Superior recovery accuracy compared to existing methods
- Robustness to noise and missing cascade observations
- Scalability to networks with thousands of nodes

## Significance
Advances the state-of-the-art in network inference from diffusion data, with applications in epidemiology, social network analysis, and infrastructure monitoring.
